### 安装 & 使用

1. 通过 npm /yarn

```bash
# yarn add xijs
npm install xijs
```

2. 通过 cdn 引入

```html
<script src="https://cdn.jsdelivr.net/npm/xijs@1.1.1/dist/xijs.min.js"></script>
```
