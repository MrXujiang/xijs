## arrayRepeat

> 计算数组中重复元素以及重复元素的次数

Demo:

```tsx | pure
import { arrayRepeat } from 'xijs';
const arr = ['1', '2', '1', 1, 2, 3, 2, 4, 5, 1]
console.log("" + arr +": ", arrayRepeat(arr))
```
